LOL Bot Privacy Policy

User data is stored in a database when you use a lol bot command, server data is stored when you use a lol bot command in that server. This data can be cleared from the database with ;removemydata and ;removeserverdata (server data removal only available for server owner). By using a command your message content data of using the command and any responses that the command asks for is also read, but this is not stored (except for counting, which can be removed with the server data removal command and only the user id and number of the last sent message of the counting channel is stored) Sniping data storage for the server is opt-in with ;enablesnipe and can be cleared with ;disablesnipe (default for sniping is set to disabled)
